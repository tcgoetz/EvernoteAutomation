Installing
- Download the release zip file.
- Double click on the work flow files in Finder to install.
- Make sure you have the required Python modules installed
-- Open a Terminal window.
-- cd into the directory of the unzipped release file.
-- Run the following command: pip install --upgrade --requirement scripts/requirements.txt

Using it
- When in Evernote choose Evernote->Services->Note to Simplified HTML from the menu to copy the current note to the clipboard as HTML.
